Keunggulan

Kenapa Pilih TeknoMedis?

🔒
Data 100% di RS/Klinik
Database SQLite tersimpan lokal di PC RS/Klinik. Sesuai Permenkes 269/2008. Kami tidak pernah menyimpan data pasien atau transaksi.
🏥
Kontinuitas Operasional
Internet mati? Sistem tetap jalan. Server kami down? Data ada di jaringan RS/Klinik Anda. Lisensi aktif sampai masa berlaku habis.
🖥️
Multi-Unit LAN
Install 1× di PC server. Semua unit (UGD, poli, rawat inap, farmasi, lab) akses via browser dari PC atau HP di jaringan lokal.
📊
Laporan Profesional
Download laporan Excel siap cetak — lengkap dengan kop RS/Klinik, tanda tangan digital direktur, nomor dokumen, format audit-ready.
🛡️
Ownership Data
Anda pemilik data — bukan kami. Stop berlangganan pun, database SQLite tetap di PC RS/Klinik. Tidak ada lock-in, tidak ada tebusan data.
🔐
Hardware-Bound License
Lisensi terikat HWID (hardware ID) PC server RS/Klinik. Tidak bisa di-copy ke komputer lain. Satu RS/Klinik, satu lisensi.
🔄
Auto-Update Aman
Fitur baru otomatis terkirim ke aplikasi Anda. Yang dikirim cuma software — bukan data. Database Anda tidak pernah di-upload kemana-mana.
💬
Dukungan Langsung
Tim support via WhatsApp — instalasi, konfigurasi, troubleshooting. Remote support dikontrol langsung oleh tim IT RS/Klinik Anda.
⚖️
Compliance-Ready
Model on-premise membuat RS/Klinik tetap memegang kendali data rekam medis. Siap audit Kemenkes, KARS, dan persyaratan JCI.

🛡️ Janji Privasi TeknoMedis

Kami tidak pernah, tidak akan, dan tidak bisa mengakses data pasien, transaksi, inventori, atau dokumen internal RS/Klinik Anda. Server kami hanya memegang lisensi. Kalau server kami di-hack sekalipun — data RS/Klinik Anda tidak terkena.

Download & Mulai

4 Langkah Mulai Pakai

Download Hub dulu sebagai launcher — gratis. Produk bisa diexplore sebelum beli lisensi.

🏗️
LANGKAH 1 — WAJIB DULU
TeknoMedis Hub
Aplikasi pusat untuk beli lisensi (pembayaran Midtrans) dan launcher semua produk yang sudah terinstall. Install 1× di PC server RS/Klinik.
⬇ Download Installer (.exe) ZIP cross-platform (24K)
Windows self-contained · no Python needed
1
🏗️ Install Hub
Jalankan installer atau extract ZIP lalu install.bat → start.bat. Buka localhost:8050 di browser.
2
💳 Beli dari Hub
Di tab Store, pilih produk + paket — bayar via Midtrans. License key otomatis dikirim ke email, terikat HWID PC.
3
⬇ Download Produk
Download ZIP produk (MedOps/UnitCost/Kalibrasi) dari halaman produk masing-masing. Install dengan installer yang sama.
4
✓ Aktivasi & Jalan
Paste license key di aplikasi, atau buka via Hub → tab Lisensi Saya. Siap dipakai semua unit di LAN klinik.
📌 Penting: Hub + produk gratis di-download & di-install. Anda bisa eksplor fitur sebelum beli lisensi. Saat dijalankan, semua data RS/Klinik tersimpan lokal di PC Anda — tidak pernah ke server kami.
🛡️ Windows Defender: Saat pertama buka TeknoMedisHubSetup.exe, Windows mungkin tampil "Windows protected your PC". Ini normal untuk installer baru yang belum code-signed. Klik "More info" → "Run anyway" untuk melanjutkan.
